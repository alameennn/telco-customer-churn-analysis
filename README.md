# Customer Churn Analysis & Retention Strategy

## Objective

The objective of this project is to analyze telecom customer behavior, identify factors influencing customer churn, and build a predictive model to estimate customer retention risk.

## Dataset

* 7043 customer records
* 33 variables
* Telecom customer service and billing information

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Project Workflow

1. Data Cleaning & Preparation
2. Exploratory Data Analysis (EDA)
3. Statistical Validation (Chi-Square Test)
4. Predictive Modeling
5. Model Evaluation
6. Business Recommendations

## Models Used

* Logistic Regression (Final Model)
* Random Forest (Comparison Model)

## Final Results

* Accuracy: 80%
* ROC–AUC: 0.849

## Key Insights

* Month-to-month customers showed highest churn
* Higher monthly charges increased churn probability
* Customers with shorter tenure churned more
* Online Security and Tech Support improved retention

## Business Recommendations

* Encourage long-term contracts
* Improve retention during initial customer lifecycle
* Promote service bundles
* Encourage automatic payment methods

Logistic Regression was selected as the final model due to better recall, strong ROC–AUC performance, and easier business interpretation.
